# Nium x Las Vegas Poker Training Proposal

Premium one-page proposal microsite for the Nium executive poker training experience at The Venetian Poker Room.

## Structure

- `index.html` — proposal page content and section architecture
- `assets/css/styles.css` — Nium-inspired visual system and responsive layout
- `assets/js/main.js` — navigation, lightbox, pricing toggle, flip cards, GSAP scroll interactions

## Deployment

This is a static site and can be deployed directly to Vercel from the repository root. No build command is required.

Suggested Vercel settings:

- Framework Preset: Other
- Build Command: leave blank
- Output Directory: leave blank
- Install Command: leave blank
